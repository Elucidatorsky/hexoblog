类型

JavaScript一共有8种主要类型。

string number boolean undefined null BigInt Symbol object

基本类型（存储在栈中）

string：字符串类型

number：数字类型

boolean：布尔类型

undefined：原始值undefined

null：null类型

BigInt：大数类型

Symbol：symbol类型

引用类型（存储在堆中）

Object：对象

Function：函数

Array：数组

Date：日期

RegExp：正则表达式

Symbol

Symbol是一种在ES6新添加的数据类型。本质上是一种唯一标识符，可用作对象的唯一属性名。

定义方法

let id = Symbol('id') //在创建symbol类型数据时参数只是为了作为标识符使用，可以不加参数

特性

1.唯一性

2.隐藏性（一般情况下无法被访问，可以通过Object.getOwnPropertySymbols()访问）

BigInt

BigInt数据类型用来表示大于2^53-1的整数。

使用方法

1.在结尾处加n

2.调用BigInt()构造函数

判断类型

1. typeof

对于基本类型，typeof能判断除了null以外的基本类型。

对于引用类型，typeof只能识别function类型，其他均返回object类型。

对于null，返回object类型。

原理

typeof是操作符而不是函数，用来检测变量的类型

数据在底层表示为二进制，js中前三位代表数据类型

typeof null为object是因为null是二进制和object相同，所以也被判断为object类型

2. instanceof

instanceof不能判断一个对象具体属于那种实例。只能给出一个对象是否为另一个对象的实例。

因为在原型链中最后都指向object类型，所以引用类型使用instanceof判断为object都能返回true。

而且在js中原型链是可以改动的，所以并不准确

原理

instanceof实际上是判断A是否为B的实例（A instanceof B），如果A是B的实例，则返回true，否则返回false。该语法等同于B.prototype.isPrototypeOf(A)

代码实现

3.constructor

当一个函数被定义时，js引擎会为函数添加[[prototype]]原型，然后在原型上添加一个constructor属性，并让其指向他的构造函数。(当任意一个普通函数用于创建一类对象时，它就被称作构造函数，或构造器。)

constructor属性的作用是，可以得知某个实例对象到底是哪一个构造函数产生的。我们可以用他来判断除了null和undefined以外的类型（null和undefined是无效的对象，不存在constructor）

但是constructor属性容易被改变，当我们重写prototype之后，被重写的constructor会丢失，会默认为Object。

4.Object.prototype.toString.call()

toString()是Object原型对象上的一个方法，该方法默认返回其调用者的具体类型，更严格的讲，是 toString运行时this指向的对象类型，返回的类型格式为[object,xxx],xxx是具体的数据类型。

可以判断任意类型。

类型转换

强制类型转换

转换为string

String(xxx); xxx.toString();

null=>”null”

undefined=>”undefined”

true=>”true” false=>”false”

10=>”10” 1e3=>”1000” 1e21=>”1e+21” 3n=>”3” // 转换Bigint不会有精度丢失

[1,2,3]=>”1,2,3”

Symbole(“xxx”)=>”Symbole(xxx)”

Object对象=>”[object Object]”

其他引用类型=>直接转换成字符串输出

转换为number

Number(xxx); parseIint(xxx); parseFloat();

null=>0

undefined=>NaN

true=>1 false=>0

字符串=>从前往后转换，碰到非数字停止转换。转换部分转换为数字，其他不转换。如果非数字开头转换为NaN。空字符串转换为0。

Date类型=>使用valueOf方法，返回从 1970 年 1 月 1 日午夜开始计的毫秒数

其他引用类型=>遵循ToPrimitive规则。

转换为boolean

Boolean(xxx)

null，undefined，NaN，空字符串“” ，0，0n转换为false；

其他都转换为true

隐式类型转换

ToPrimitive规则

引用类型转换为原始类型的操作。

原始类型[基本类型、基本数据类型、原始数据类型]是一种既非对象也无方法的数据。在 JavaScript 中，共有7种：string，number，bigint，boolean，null，undefined，symbol

ToPrimitive(obj,type);

obj是需要转换的值，type是可选的期望类型。转换后的结果由期望类型决定。

type为string:

先调用obj的toString方法，如果为原始值，则return，否则进行第2步
调用obj的valueOf方法，如果为原始值，则return，否则进行第3步
抛出TypeError 异常

type为number:

先调用obj的valueOf方法，如果为原始值，则return，否则进行第2步
调用obj的toString方法，如果为原始值，则return，否则第3步
抛出TypeError 异常

type参数为空

该对象为Date，则type被设置为String
否则，type被设置为Number

valueOf方法

Array返回数组对象本身

Boolean返回布尔值

Date返回从 1970 年 1 月 1 日午夜开始计的毫秒数

Function返回函数本身

Number返回数字值

Object返回对象本身

String返回字符串值

Math和Error对象没用valueOf方法

== 的过程（优先换成数字、字符串）

  1. 首先看==前后有没有NaN，有的话都是返回false。NaN不等于任何值，包括其本身
  2. 布尔值会转成数字类型，true转成1，false转成0
  3. 数字和字符串比较，字符串会转成数字
  4. undefined和null除了和undefined或null相等，和其他相比都是false
  5. 数字或者字符串和对象相比，对象使用ToPrimitive规则转换。
  6. 当两个操作数都是对象时，JavaScript会比较其内部引用，当且仅当他们的引用指向内存中的相同对象（区域）时才相等，即他们在栈内存中的引用地址相同。

+的过程（优先换成字符串、数字）

  1. 如果至少有一个操作数是对象，它会被转换成原始值（字符串，数字或布尔）；
  2. 转换之后，如果至少有一个操作数是字符串类型，第二个操作数会被转换成字符串，并且会执行连接。
  3. 在其他的情况下，两个操作数都会转换成数字并执行算数加法运算。
